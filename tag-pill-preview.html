tagPillStyle() — brand palette + hash fallback (dark / light)

Two-tier resolution. First, a curated brand palette with ~250 dev/infra entries is consulted with case- and hyphen/underscore/whitespace-insensitive lookup (back-end, Back End, BACKEND → same key). Unmatched tags fall through to the hash-based path with curated hue bands (purples skipped, blues widened) and a ~20% chance of rendering as a near-neutral grey. Iconic brand purples (Terraform, Vite, Bootstrap, PHP, …) are allowed in the override only. The theme parameter (default "dark") selects the lightness ramp — use the buttons above to flip.

Self-contained preview — the palette and algorithm are inlined below so this file renders without depending on the production TS build. When src/utils/tag-pills.ts changes, refresh the JS palette inside this HTML to match.
